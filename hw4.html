List Usage
Unordered Lists An unordered list is shown, by default, with a bullet – those lists are generally used to present a list of items that do not necessarily need to be shown in any specific order. To start this list use the tag <ul>.
Ordered List An ordered list is shown, by default, with a number – those lists are generally used to present a list of items that need to be shown in a specific order. To start this list use the tag <ol>.
Definition Lists In a deffintion list items come in pairs – the term to be defined and the definition (description). This listis used for glossary, vocabulary, questions and answers, etc. To start this list use the tag <dl>.
